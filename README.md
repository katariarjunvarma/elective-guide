# Elective Recommendation System

This project is a web-based Elective Recommendation System that helps university students choose suitable elective courses based on their academic profile, interests, and career goals.

## Getting Started

### Prerequisites

- Node.js and npm installed on your machine.

### Installation

```sh
npm install
```

### Development

Start the development server:

```sh
npm run dev
```

Then open the URL printed in the terminal (typically `http://localhost:5173/`) in your browser.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Project Overview

The Elective Recommendation System provides:

- A clean UI for students to input their interests and academic details.
- Recommendations for elective courses based on configurable criteria.
- A responsive layout suitable for desktop and mobile devices.
